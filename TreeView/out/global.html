branch_hover_cb()

This Callback is triggerd when the user hovers over or away from a branch. This callback takes 3 parameters
Parameters:
Type Description
string The uid of the branch that the hover pertains to
object The user object associated with the pertaining branch
boolean True if user hovers over a branch, false if user hovers away from a branch

branch_rightclick_cb()

This Callback is triggerd when the user right-clicks on a branch. Can be used to render a context specific list of options. This callback takes 4 parameters
Parameters:
Type Description
string The uid of the branch that the right-click pertains to
object The user object associated with the pertaining branch
number X coordinate of the mouse position at the time (Left of screen is at 0)
number Y coordinate of the mouse position at the time (top of screen is at 0)

branch_selected_cb()

This Callback is triggered when the user selects a branch either by clicking or by double clicking on it. The callback takes 3 parameters.
Parameters:
Type Description
string The uid of the branch that was selected
object The user object associated with the selected branch
boolean True if user double clicked on branch, false if single clicked
